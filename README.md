# voicebase-player-v2
The react-based version of the VoiceBase Player

## Requirements
* node `>=5.0.0`
* npm `^3.0.0`

## Development
**Install**

```
npm install
```

**Start the application in dev mode with hot-module-replacement**

```
npm start
```

**Build for production**

```
npm run build
```

**Run tests**

```
npm run test
```
